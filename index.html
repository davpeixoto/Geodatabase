Sobre o Projeto

O GEODatabase Brasil é uma plataforma colaborativa de dados geoespaciais desenvolvida por bolsistas do CNPq no Instituto Federal de Brasília.

Objetivo

Apresentar a construção de uma base de dados geomorfométrica nacional em alta resolução (1:100.000), integrada em um GEODatabase colaborativo, para subsidiar análises de risco geomorfológico, planejamento territorial e políticas de prevenção de desastres no Brasil.

Conceitos-chave
Risco geomorfológico

Probabilidade de ocorrência de processos naturais (deslizamentos, enchentes) em áreas ocupadas.

GEODatabase Brasil

Plataforma que integra dados espaciais de alta resolução para análise multiescalar.

Metodologia

A base foi construída a partir de MDE hidrologicamente consistente, com derivadas de primeira e segunda ordem (declividade, orientação, amplitude) em escala 1:100.000.
